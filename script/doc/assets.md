# 战利品处理

战利品处理是由预设，任务设定，用户变量设置共同决定的

预设的撇之文件位于 script/data/assets文件

变量在游戏文件的sell变量里

格式一致，都是一行一个配置，格式为

```
#处理方式 判断类型=条件列表
```

## 处理方式

脚本支持的处理方式如下

* #home 存入房子的聚宝箱
* #carry 带在身上
* #drop  丢弃,实际实现为给扬州客店小二
* #opananddrop 打开后丢弃，用于处理freequest的包裹
* #drophere 就地丢弃，一般不使用
* #pack 放布袋里

特别的，对于 #home #carry 处理的物品，师门任务的奖励采用自动模式时，会answer Y,其他的会answer N  

## 判断类型

默认的判断类型为下

* id 通过id完整匹配，不区分大小写
* name 名字完整匹配
* start 名字以条件开始的，id中必须含有空格
* end 名字以条件结束的，id中必须含有空格
* contain 名字包含条件的，，id中必须含有空格
* any 任意
